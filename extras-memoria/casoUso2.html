Caso de Uso: Solicitar Ayuda (UC-02)

ID del Caso de Uso UC-02
Nombre del Caso de Uso Solicitar Ayuda (/help)
Actor Principal Usuario de Telegram
Descripción Permite al usuario conocer los comandos disponibles del bot y recibir una breve descripción de cómo utilizarlos.
Precondiciones
  • El bot debe estar activo y accesible en Telegram.
Postcondiciones
  • El usuario obtiene una lista de comandos y una explicación de cada uno.
Curso Normal
  1. El usuario envía el comando /help.
  2. El sistema (bot) valida que esté disponible.
  3. El sistema responde con un mensaje que describe los comandos principales (ejemplo: /start, /register, /recordatorios, /clima).
Flujos Alternativos
  • F1: Bot no disponible
    • El usuario no recibe respuesta si el bot está inactivo.
    • Podrá reintentar más tarde.
Excepciones
  • E1: Error de conexión
    • Fallo de conexión con la API de Telegram u otra incidencia.
    • El usuario no recibe el mensaje de ayuda.
