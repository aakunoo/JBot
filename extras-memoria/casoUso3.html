Caso de Uso: Registrar Usuario (UC-03)

ID del Caso de Uso UC-03
Nombre del Caso de Uso Registrar Usuario (/register)
Actor Principal Usuario de Telegram
Descripción Permite al usuario darse de alta en el sistema, asociando su chat_id y posibles datos adicionales (username, fecha de registro, etc.) en la base de datos. Este comando habilita el uso de funcionalidades avanzadas como la gestión de recordatorios y la consulta de clima.
Precondiciones
  • El bot debe estar en funcionamiento y accesible en Telegram.
Postcondiciones
  • El usuario queda registrado en la base de datos (colección "usuarios").
  • Se habilita el acceso a funcionalidades que requieren registro (/recordatorios, /clima, etc.).
Curso Normal
  1. El usuario envía el comando /register.
  2. El sistema comprueba si el usuario ya está registrado.
  3. Si no lo está, el sistema crea un nuevo registro en la base de datos asociando el chat_id del usuario, username, fecha/hora de registro, etc.
  4. El sistema confirma el registro y notifica al usuario que ya puede acceder a las funciones avanzadas.
Flujos Alternativos
  • F1: Usuario ya registrado
    • El sistema detecta que el usuario ya posee un registro en la base de datos.
    • Se muestra un mensaje indicando que ya está registrado.
    • Fin del caso de uso.
Excepciones
  • E1: Error de conexión o escritura en la base de datos
    • Si no se puede completar el registro por problemas de infraestructura, se notifica al usuario.
    • El usuario puede intentar de nuevo más tarde.
