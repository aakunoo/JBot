Caso de Uso: Iniciar Bot (UC-01)

ID del Caso de Uso UC-01
Nombre del Caso de Uso Iniciar Bot (/start)
Actor Principal Usuario de Telegram
Descripción Permite al usuario iniciar o reiniciar la interacción con el bot. Normalmente, el bot envía un mensaje de bienvenida y muestra las funcionalidades disponibles o un breve resumen de sus comandos.
Precondiciones
  • El bot debe estar activo y accesible en Telegram.
Postcondiciones
  • Se inicia la conversación con el bot y el usuario puede acceder a sus comandos.
Curso Normal
  1. El usuario envía el comando /start al bot.
  2. El sistema (bot) responde con un mensaje de bienvenida o introducción.
  3. Opcionalmente, se muestra un menú inicial o se listan los comandos disponibles.
Flujos Alternativos
  • F1: El bot no está disponible
    • Si el bot está inactivo o no se puede acceder, el usuario no recibe respuesta.
    • El usuario podría reintentar más tarde o verificar su conexión.
Excepciones
  • E1: Error de conexión
    • Si existe un problema de red o de la API de Telegram, no se envía el mensaje de bienvenida.
    • El usuario deberá reintentar cuando el servicio esté operativo.
